# Verificador de Batimentos Cardíacos

Este programa tem como objetivo verificar se os batimentos cardíacos por minuto (BPM) de uma pessoa estão **dentro**, **abaixo** ou **acima** do recomendado para a sua idade.

---

## 📌 Funcionalidades
- Recebe como entrada:
  - Batimentos cardíacos por minuto (BPM)
  - Idade do usuário
- Compara os valores informados com faixas de referência.
- Informa se os batimentos estão:
  - Dentro do intervalo recomendado
  - Abaixo do recomendado
  - Acima do recomendado
